# Resources
{: .no_toc }

The Resources section serves as a comprehensive library for technical documentation related to Audio and Moving Image preservation. We have organized these resources into three primary categories to help you find exactly what you need.

## [Computer Setup]({% link pages/resources/computer-setup/computer-setup-overview.md %})
Essential documentation for configuring your local environment.
* **macOS Terminal:** Setup instructions and best practices.
* **Environment Variables:** Understanding and modifying your PATH.
* **Troubleshooting:** Resolving JDBC and connectivity issues.

## [Technical Tools]({% link pages/resources/technical-tools/technical-tools-overview.md %})
Guides for the specific software and utilities used in our daily operations.
* **CLI Resources:** General command-line tips.
* **Tool Guides:** Instructions for `tmux`, `ssh`, and `rclone`.

## [Workflow Resources]({% link pages/resources/workflow-resources/workflow-resources-overview.md %})
NYPL-specific workflows and procedural documentation.
* **ServiceNow:** Navigating Digital Access (DAR) and Media Digitization (MDR) requests.
* **Digitization:** Qumolo workflows and AMI file specifications.
* **Migration:** Managing statuses and troubleshooting SPEC system issues.
